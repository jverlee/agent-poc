# Generate Social Media Image

## Description
This skill generates social media images optimized for various platforms (Instagram, Twitter/X, LinkedIn, etc.).

## Usage
Provide a prompt describing the desired image, and this skill will generate a platform-ready social media graphic.

## Parameters
- `platform`: Target social media platform (e.g., instagram, twitter, linkedin)
- `prompt`: Description of the image to generate
- `style`: Visual style (e.g., minimal, bold, professional)

## Output
A generated image file ready for posting.
