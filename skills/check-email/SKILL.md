# Check Email

## Description
This skill checks an email inbox for new messages and can summarize or filter them.

## Usage
Invoke this skill to scan the inbox and return a summary of recent or unread emails.

## Parameters
- `filter`: Filter criteria (e.g., unread, from, subject)
- `limit`: Maximum number of emails to return
- `summarize`: Whether to generate a summary of each email (true/false)

## Output
A list of matching emails with optional summaries.
