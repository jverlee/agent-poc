# Web Search

## Description
This skill performs web searches and returns structured results with titles, snippets, and URLs.

## Usage
Invoke this skill to search the web for information, news, documentation, or any topic.

## Parameters
- `query`: The search query string
- `limit`: Maximum number of results to return
- `recency`: Filter by time range (e.g., day, week, month, any)

## Output
A list of search results with titles, URLs, and content snippets.
